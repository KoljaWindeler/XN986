SONIX SN986 Series IP Camera SoC

Overview
The SONiX SN98600 / 98601 / 98610 IP Camera SoC integrates powerful image sensor processing, FHD 1080p30 H.264 multi-stream encoding, and ARM9 processor with rich I/O for IP Camera and network video stream server applications. SN98600 / 98601 / 98610 offers excellent video quality and supports varied real-time bitstreams, up to 2 simultaneous streams with different video formats (H.264 and MJPEG), and different resolutions to fit the bandwidth.
SONiX SN98600 / 98601 / 98610 provides fully-featured IP Camera turn-key solution for customers’ development, including network video stream server, customizable AWB/AE/AF, SONiX download/ISP/debug tools …etc.
